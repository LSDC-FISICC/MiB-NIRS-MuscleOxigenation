Detailed Description

16-bit unsigned integer representation of ADC counts

Intermediate format after combining 2 raw bytes per channel. Range: 0 to 65535 (16-bit ADC counts) Represents: Photodiode current as raw ADC values

Note
Use MAX30101_ConvertSampleToUint16() to convert from raw samples
See also
MAX30101_ConvertSampleToUint16, MAX30101_ReadFIFO_Current
