Detailed Description

Raw FIFO sample data directly read from sensor (6 bytes) in multi-LED mode.

Intermediate storage for raw 2-byte ADC values from each LED channel. Format: 3 channels × 2 bytes (MSB, LSB) per sample Total: 6 bytes per complete sample in multi-LED mode

Note
Use MAX30101_ReadFIFO() to populate this structure
