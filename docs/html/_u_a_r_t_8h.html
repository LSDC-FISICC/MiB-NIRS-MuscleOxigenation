void UART_Config ( uint32_t baud_rate)

Initialize USART2 for configurable baud rate transmission.

Configuration sequence:

  1. Enable clocks: USART2, GPIOA
  2. Configure PA2 (TX) and PA15 (RX) as alternate function AF7
  3. Configure USART2: desired baud, 8-bit data, 1 stop bit
  4. Enable receiver interrupt
Parameters
baud_rate- Desired baud rate
Returns
void
Return values
N/A
Timing:
  • Total init time: <1 ms
  • USART2 ready after: function completes
Warning
  • Call once at system startup, BEFORE any USART2_Send operations
  • External USB-UART converter required on PA2/PA15
See also
USART2_Send, USART2_putString

Complete USART2 setup sequence:

  1. Enable GPIOA and USART2 clocks
  2. Configure PA2 (TX) and PA15 (RX) as AF7 (Alternate Function 7)
  3. Configure BRR for desired baud rate
  4. Enable transmitter, receiver, and receiver interrupt

void USART2_putString ( char * string)

Send null-terminated string via UART.

Transmits string character-by-character using USART2_Send()

Parameters
string- Pointer to null-terminated character string
Returns
void
Return values
N/A(blocking)
Timing:
  • Per-character latency: ~22 µs at 460800 baud (10 bits/byte: 8N1)
  • For 10-char string: ~220 µs; typical CSV frame ~350 µs
Data Format:
  • String must be null-terminated
  • Each character transmitted as 8-bit value
Usage Example:
USART2_putString("Hello UART\r");
