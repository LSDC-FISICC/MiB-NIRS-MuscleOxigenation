Initialize MAX30101 in Sp\+O2 mode (dual-\/\+LED:\+ Red + IR) Configures sensor for blood oxygen (Sp\+O2) measurement with low power consumption.
\begin{DoxyItemize}

\item ADC Range:\+ 2048 nA full-\/scale (SPO2\+\_\+\+CONFIG bits \+[6:\+5]\+ = 00)
\item FIFO Configuration:\+ No averaging, rollover enabled
