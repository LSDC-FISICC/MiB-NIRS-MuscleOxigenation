Initialize USART2 for configurable baud rate transmission. 

Complete USART2 setup sequence:\+
\begin{DoxyEnumerate}
\item Enable GPIOA and USART2 clocks
\item Configure PA2 (TX) and PA15 (RX) as AF7 (Alternate Function 7)
\item Configure BRR for desired baud rate
\item Enable transmitter, receiver, and receiver interrupt
\end{DoxyEnumerate}
